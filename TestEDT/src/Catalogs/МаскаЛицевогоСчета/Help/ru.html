Справочник доступен из меню: «Прочая информация» → «См. также» → «Маски лицевого счета».

В карточке маски имеются поля:

§   Наименование – произвольное название для идентификации объекта в Системе.

§   Маска представления лицевого счета - возможность указывать формат представления лицевого счета вида. Если не заполнена, то маской по умолчанию является строка «*****.***.*.*.**********», в которой разделителями являются точки. Пользователь может указать свою строку представления, убрать или добавить точки.

§   Шаблон – запись на языке 1С для настройки программного формирования кода лицевого счета.

Пример шаблона:

Результат = Контекст.Счет.Код  + Справочники.Валюты.КодВалютыДляЛицевогоСчета(Контекст.Валюта) + Справочники.Портфели.ПризнакДоверительногоУправления(Контекст.Портфель) + Справочники.Портфели.ПризнакВидаДеятельностиНПФ(Контекст.Портфель) + ДополнитьЛидирующимиНулями(ЗначениеРеквизитаАналитики(Контекст, "Договоры", "КодДляЛицевогоСчета"), 5) + ДополнитьЛидирующимиНулями(Контекст.ПорядковыйНомер, 5);

 

При вводе шаблона может использоваться функция ДополнитьЛидирующимиНулями(Значение, Длина) - возвращает строку заданной длины. Если значение короче длины, возвращаются лидирующие нули плюс значение. Если значение длиннее длины, то возвращается обрезанное с начала значение.

В маске могут принимать участие коды (будет взято значение из реквизита «КодДляЛицевогоСчета») из следующих справочников: «Портфели», «Договоры контрагентов», «Банковские счета», «Акции», «Облигации».

В табличной части необходимо установить вид аналитики, использующийся в тексте шаблона. Например, если задан шаблон, приведенный выше, то в табличной части следует установить аналитику «Договоры контрагента». Такой шаблон сможет применяться только для счетов ЕПС, в аналитике которых заполняется договор контрагента, например, для счета 43218. Совпадение аналитик контролируется Системой в момент записи маски в регистр «Настройка аналитики лицевых счетов ЕПС», при несовпадении будет выведено сообщение об ошибке.

После нажатия кнопки «Записать и закрыть», Система сохраняет введенные данные в справочнике «Маски лицевого счета» и закрывает окно ввода информации.
