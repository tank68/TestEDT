Документ доступен из меню: «Операции» → «Операции по портфелю» → «Операции, введенные вручную».

Документ предназначен для ручной корректировки данных бухгалтерского учета по планам РСБУ и ЕПС в соответствующих учетных регистрах. Также можно вносить корректирующие записи в регистры накопления и регистры сведений. Список доступных регистров открывается по кнопке «Выбор регистров».
