Отчет предназначен для представления данных по оборотам между выбранным счетом и всеми остальными счетами за определенный период по указанному портфелю.

Отчет показывает, в каком объеме и в корреспонденции с какими счетами поступали и были списаны средства на выбранном счете в течение указанного периода времени. Отчет можно использовать как компактную выборку данных по указанному счету из объемных отчетов «Оборотно-сальдовая ведомость ЕПС» и «Шахматная ведомость ЕПС».

Отчет можно сформировать с детализацией по субсчетам или по объектам аналитического учета (субконто). Обороты по дебету и кредиту можно детализировать до субсчетов корреспондирующих счетов.

Данные можно вывести с дополнительной разбивкой по периодам времени: месяц, год и т.д.

В отчете можно установить следующие настройки:

§   Группировка – предусмотрена возможность выбора параметров, по которым будет происходить объединение отображаемых в отчете данных в группы (в табличной части вкладки). Флаги «По субсчетам» и «По субсчетам кор.счетов» позволяет быстро настроить соответствующие группировки данных.

§   Отбор – при необходимости можно настроить фильтр отображаемых в отчете данных, указав значения полей, по которым будет происходить отбор.

§   Показатели – необходимо отметить флагами поля, которые должны выводиться в отчете. Доступность показателей зависит от выбранного в «шапке» счета ЕПС.

§   Дополнительные поля – по клику кнопки «Добавить» при необходимости можно настроить отображение в отчете дополнительных полей и их значений.

§   Сортировка – по клику кнопки «Добавить» при необходимости можно указать параметры, по которым произойдет упорядочивание выводимых в отчете данных в том или ином направлении.

§   Оформление – при необходимости можно указать значение настроек, влияющих на визуальное представление данных, выводимых в отчете. «Вариант оформления» характеризует цвет заливки, также предусмотрена возможность выбора параметра оформления отчета и его значения (в табличной части вкладки).

По клику кнопки «Сформировать» Система построит и выведет на экран отчет согласно указанным настройкам.

По команде «Сохранить настройки» происходит сохранение варианта построения отчета согласно текущим настройкам. По клику «Выбрать настройки» открывается форма, в которой можно выбрать одну из ранее сохраненных вариантов настроек.
