Отчет предназначен для представления данных по выбранному виду субконто: начальное и конечное сальдо, обороты за период по счетам. Отчет можно сформировать не только по выбранному субконто или нескольким субконто, но и по значению субконто или реквизиту значения субконто.

Отчет формируется с детализацией по счетам. Настройки отчета позволяют задать дополнительную детализацию по субсчетам.

Данные можно вывести с разбивкой по периодам времени: месяц, год и т.д.

В отчете можно вывести развернутое сальдо. При этом развернутое сальдо рассчитывается для каждого уровня группировки.

В отчете можно установить следующие настройки:

§   Группировка – по клику кнопки «Добавить» необходимо указать параметры, по которым будет происходить объединение отображаемых в отчете данных в группы. Флаг «По субсчетам (ЕПС)» позволяет быстро настроить группировку по субсчетам.

§   Отбор – по клику кнопки «Добавить» при необходимости можно настроить фильтр отображаемых в отчете данных, указав значения показателей, по которым будет происходить отбор.

§   Виды субконто – на вкладке дублируется информация о выбранных видах субконто в «шапке» отчета, по которым происходит его формирование. Также по клику кнопки «Добавить» при необходимости можно расширить количество видов.

§   Показатели – необходимо отметить флагами поля, которые должны выводиться в отчете. Доступность флагов зависит от выбранного в «шапке» субконто ЕПС.

§   Дополнительные поля – по клику кнопки «Добавить» при необходимости можно настроить отображение в отчете дополнительных полей и их значений.

§   Сортировка – по клику кнопки «Добавить» при необходимости можно указать параметры, по которым произойдет упорядочивание выводимых в отчете данных в том или ином направлении.

§   Оформление – при необходимости можно указать значение настроек, влияющих на визуальное представление данных, выводимых в отчете. «Вариант оформления» характеризует цвет заливки, также по клику кнопки «Добавить» предусмотрена возможность выбора параметра оформления отчета и его значения.

По клику кнопки «Сформировать» Система построит и выведет на экран отчет согласно указанным настройкам.

По клику кнопки «Сохранить настройки» происходит сохранение варианта построения отчета согласно текущим настройкам. По клику «Выбрать настройки» открывается форма, в которой можно выбрать одну из ранее сохраненных вариантов настроек.
