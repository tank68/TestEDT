Отчет предназначен для формирования оборотно-сальдовой ведомости по всем счетам (субсчетам) за определенный период времени по выбранному портфелю.

Каждая строка отчета соответствует определенному счету или субсчету. Счета упорядочены в порядке возрастания.

Для каждого счета показаны суммы остатков на начало и на конец периода по дебету и кредиту и суммы оборотов за период.

В отчете можно установить следующие настройки:

§   Группировка – по клику кнопки «Добавить» при необходимости можно указать счета, по которым будет происходить объединение отображаемых в отчете данных в группы. Флаги «По субсчетам (ЕПС)» и «По лицевым счетам» позволяют быстро настроить соответствующие группировки для всех счетов.

§   Отбор – по клику кнопки «Добавить» при необходимости можно настроить фильтр отображаемых в отчете данных, указав значения показателей, по которым будет происходить отбор. Также предусмотрена настройка отображения информации по забалансовым счетам.

§   Показатели – необходимо отметить флагами поля, которые должны выводиться в отчете.

§   Развернутое сальдо – предусмотрена возможность настройки отображения развернутого сальдо для указанных счетов (добавление в табличную часть происходит по клику кнопки «Добавить»). Начальное и конечное сальдо по выбранным счетам будут выведены в развернутом виде.

§   Дополнительные поля – по клику кнопки «Добавить» при необходимости можно настроить отображение в отчете дополнительных полей и их значений.

§   Оформление – при необходимости можно указать значение настроек, влияющих на визуальное представление данных, выводимых в отчете. «Вариант оформления» характеризует цвет заливки, также по клику кнопки «Добавить» предусмотрена возможность выбора параметра оформления отчета и его значения.

По клику кнопки «Сформировать» Система построит и выведет на экран отчет согласно указанным настройкам.

По клику кнопки «Сохранить настройки» Система сохраняет вариант построения отчета согласно текущим настройкам. По клику «Выбрать настройки» открывается форма, в которой можно выбрать одну из ранее сохраненных вариантов настроек.
