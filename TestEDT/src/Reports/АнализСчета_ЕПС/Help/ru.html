Отчет предназначен для представления данных по оборотам между выбранным счетом и всеми остальными счетами за определенный период.

По содержанию выводимой информации отчет похож на отчет Обороты счета. Отличие заключается в форме представления информации.

Отчет можно сформировать с детализацией по субсчетам или по объектам аналитического учета (субконто). 

Данные можно вывести с дополнительной разбивкой по периодам времени: месяц, год и т.д.

В отчете можно вывести развернутое сальдо. При этом развернутое сальдо рассчитывается для каждого уровня группировки и по счету в целом.

В отчете можно установить следующие настройки:

§   Группировка – по клику кнопки «Добавить» необходимо указать параметры, по которым будет происходить объединение отображаемых в отчете данных в группы. Флаг «По субсчетам (ЕПС)» позволяет быстро настроить группировку по субсчетам.

§   Группировка по кор. счетам – по клику кнопки «Добавить» при необходимости можно выбрать корреспондирующие счета (для счета, указанного в «шапке» отчета), по которым будет происходить объединение отображаемых данных в группы. Также предусмотрена возможность группировки по субсчетам и субконто выбранных корреспондирующих счетов. Флаг «По субсчетам кор. счетов (ЕПС)» позволяет быстро настроить группировку по субсчетам.

§   Отбор – по клику кнопки «Добавить» при необходимости можно настроить фильтр отображаемых в отчете данных, указав значения показателей, по которым будет происходить отбор.

§   Показатели – необходимо отметить флагами поля, которые должны выводиться в отчете. Доступность показателей зависит от выбранного в «шапке» счета ЕПС.

§   Дополнительные поля – по клику кнопки «Добавить» при необходимости можно настроить отображение в отчете дополнительных полей и их значений.

§   Сортировка – по клику кнопки «Добавить» при необходимости можно указать параметры, по которым произойдет упорядочивание выводимых в отчете данных в том или ином направлении.

§   Оформление – при необходимости можно указать значение настроек, влияющих на визуальное представление данных, выводимых в отчете. «Вариант оформления» характеризует цвет заливки, также по клику кнопки «Добавить» предусмотрена возможность выбора параметра оформления отчета и его значения.

По клику кнопки «Сформировать» Система построит и выведет на экран отчет согласно указанным настройкам.
