Регистр доступен через меню: «Прочая информация» → «Бухгалтерский учет ЕПС» → «Настройка аналитики лицевых счетов ЕПС».

В регистре задаются параметры, необходимые для настройки механизма автоматического ведения 20-значных лицевых счетов. Настройки аналитики можно задавать в разрезе каждого Портфеля. При установке аналитики для одного из парных лицевых счетов – аналитика на втором парном счете установится точно такая же автоматически (т.к. происходит закрытие парных счетов, то Система контролирует совпадение аналитики на обоих счетах пары).

ВНИМАНИЕ! В Системе предустановлена стандартная настройка для счетов с пустым портфелем - то есть она распространяется на все портфели, имеющиеся в Системе. Если для какого-то портфеля необходимо внести изменения в  данный регистр, то надо СОЗДАТЬ новую запись по нужному портфелю или СКОПИРОВАТЬ существующую запись, поставить там портфель и затем её изменить. Иначе, установка портфеля в единственную существующую предопределённую запись приведёт к тому, что во всех остальных портфелях на лицевых счетах по этому счету вообще не будет вестись никакой аналитики.

В регистре сведений содержатся следующие данные:

Измерения:

§   Счет – выбор счета из единого плана счетов. Обязательное для заполнения поле.

§   Портфель – указывается портфель, для которого создаётся настройка. Если портфель не указан, то настройка будет применяться для всех портфелей.

Ресурсы:

§   Аналитика1, 2 … 5 – выбор из Плана видов характеристик «Виды субконто ЕПС».

Реквизиты: нет.

Пример. Если по счету 20601 (Депозиты в кредитных организациях) не установлен Портфель и установлена только Аналитика1 = «Контрагенты», то при проведении для любого портфеля по каждому новому контрагенту будет автоматически создан новый лицевой счет. Если установить ещё и Аналитика2 = «Договоры контрагента». То новый лицевой счет будет автоматически создаваться для каждой пары «Контрагент-Договор контрагента». Если установить Портфель, то для каждого Портфеля будут создаваться свои аналитические счета в соответствии с установленными аналитиками.
